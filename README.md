# ResumeFlow
ResumeFlow is a modern ATS-friendly resume builder web application designed to help users create professional resumes through an interactive and user-centric interface. The platform provides real-time resume previewing, ATS score analysis, secure user authentication, dashboard-based resume management, and print-optimized resume exporting



✨ Features
🔐 User Authentication System (Login & Signup)
📄 Dynamic Resume Builder
⚡ Real-Time Resume Preview
📊 ATS Score Calculator
💾 Resume Saving using LocalStorage
📂 Dashboard for Resume Management
📥 Resume Download & Print Support
🎨 Responsive and Modern UI
